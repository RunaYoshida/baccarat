# Baccarat(バカラ)
トランプゲームのバカラ。
参加者が「BANKER」と「PLAYER」の勝敗を予想するゲーム。

# ルール
参加者は「BANKER」「PLAYER」「タイ」のいずれかに賭ける。点数の数え方は、A~9 までは数字通りにカウント。10~K まではすべて 0 とカウントする。
繰り上がった場合、下一桁のみを評価する。

(例) 7のカードと8のカード→点数 5

勝敗の決め方は、より 9 に近い方。
プレイヤーサイド・バンカーサイド双方に2枚ずつカードが配られ、条件に従い3枚目が配られる。
